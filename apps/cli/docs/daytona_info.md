## daytona info

Get sandbox info

```
daytona info [SANDBOX_ID | SANDBOX_NAME] [flags]
```

### Examples

```
  daytona info my-sandbox
  daytona info my-sandbox --format json
```

### Options

```
  -f, --format string   Output format. Must be one of (yaml, json)
```
