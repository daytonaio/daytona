## daytona list

List sandboxes

```
daytona list [flags]
```

### Examples

```
  daytona list
  daytona list --limit 50
  daytona list --format json
```

### Options

```
  -c, --cursor string   Cursor for pagination
  -f, --format string   Output format. Must be one of (yaml, json)
  -l, --limit int       Maximum number of items per page (default 100)
```
