## daytona mcp init

Initialize Daytona MCP Server with an agent (currently supported: claude, windsurf, cursor)

```
daytona mcp init [AGENT_NAME] [flags]
```

### Examples

```
  daytona mcp init claude
  daytona mcp init cursor
```
