## daytona volume get

Get volume details

```
daytona volume get [VOLUME_ID_OR_NAME] [flags]
```

### Examples

```
  daytona volume get my-volume
  daytona volume get my-volume --format json
```

### Options

```
  -f, --format string   Output format. Must be one of (yaml, json)
```
