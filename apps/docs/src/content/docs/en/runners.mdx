---
title: Customer Managed Compute
description: Use your own runner machines to scale compute resources independently within each region.
---

Customer Managed Compute enables you to use your own runner machines to run sandbox workloads. Runners are machines that power Daytona's compute plane and provide the underlying infrastructure.

:::caution[Experimental]
This feature is experimental. To request access, contact [support@daytona.io](mailto:support@daytona.io).
:::
