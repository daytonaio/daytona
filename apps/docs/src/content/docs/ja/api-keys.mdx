title: API キー
---

Daytona の API キーは、[Daytona API](/docs/ja/tools/api) へのリクエストを認証するために使用します。

## API キーを作成する \{#create-an-api-key\}

1. [Daytona Dashboard ↗](https://app.daytona.io/dashboard/keys) にアクセスします
2. **Create Key** ボタンをクリックします
3. API キーの名前を入力し、有効期限を設定し、[権限を選択します](#permissions--scopes)
4. **Create** ボタンをクリックして API キーを作成します
5. API キーをクリップボードにコピーします

アプリケーションで API キーを使用するには、`DAYTONA_API_KEY` 環境変数を設定します。

Daytona では、環境を設定する方法が複数用意されています。[コード内](/docs/ja/configuration#configuration-in-code)、[環境変数](/docs/ja/configuration#environment-variables)、[.env ファイル](/docs/ja/configuration#env-file)、および [デフォルト値](/docs/ja/configuration#default-values) を利用できます。

## 権限とスコープ \{#permissions-scopes\}

| リソース   | スコープ                 | 説明                      |
| ---------- | ----------------------- | ------------------------- |
| Sandboxes  | **`write:sandboxes`**   | サンドボックスを作成/変更 |
|            | **`delete:sandboxes`**  | サンドボックスを削除      |
| Snapshots  | **`write:snapshots`**   | スナップショットを作成/変更 |
|            | **`delete:snapshots`**  | スナップショットを削除    |
| Registries | **`write:registries`**  | レジストリを作成/変更     |
|            | **`delete:registries`** | レジストリを削除          |
| Volumes    | **`read:volumes`**      | ボリュームを閲覧          |
|            | **`write:volumes`**     | ボリュームを作成/変更     |
|            | **`delete:volumes`**    | ボリュームを削除          |
| Audit      | **`read:audit_logs`**   | 監査ログを閲覧            |
| Regions    | **`write:regions`**     | リージョンを作成/変更     |
|            | **`delete:regions`**    | リージョンを削除          |
| Runners    | **`read:runners`**      | ランナーを閲覧            |
|            | **`write:runners`**     | ランナーを作成/変更       |
|            | **`delete:runners`**    | ランナーを削除            |
|            | `delete:runners`        | ランナーを削除            |
