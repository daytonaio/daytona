[Daytona Billing ↗](https://app.daytona.io/dashboard/billing/spending) では、組織の請求情報、ウォレット、利用状況の詳細な概要を確認できます。

Daytona では、リソースごとの費用の内訳を可視化するチャートが表示されます。面グラフと棒グラフを切り替えられ、RAM、CPU、ストレージでフィルタリングできます。また、日付範囲（過去 12 か月、6 か月、3 か月）でフィルタリングすることもできます。

## ウォレット \{#wallet\}

[Daytona Wallet ↗](https://app.daytona.io/dashboard/billing/wallet) では、組織のウォレット残高と今月のクレジット消費額が表示されます。

### 支払い方法を追加する \{#add-payment-method\}

Daytona では、ウォレットに追加できる支払い方法を利用できます。クレジットカードをウォレットに連携して、残高をチャージできます。

1. [Daytona Wallet ↗](https://app.daytona.io/dashboard/billing/wallet) に移動します
2. **Payment method** セクションの横にある **Connect** ボタンをクリックします
3. 表示される案内に従って、クレジットカードをウォレットに接続します。

組織はウォレットに対して自動チャージのルールを設定できます。

* **しきい値** — ウォレット残高がこの金額を下回ると、チャージが開始されます。
* **目標額** — ウォレット残高がこの金額まで補充されます。

自動チャージを無効にするには、**しきい値** と **目標額** の両方を `0` に設定してください。

### クーポンの利用 \{#redeem-coupon\}

Daytona では、ウォレットにクレジットを追加できるクーポンコードを提供しています。

1. [Daytona Wallet ↗](https://app.daytona.io/dashboard/billing/wallet) に移動します。
2. **Redeem coupon** 入力フィールドにクーポンコードを入力します。
3. クーポンコードを適用するには、**Redeem** ボタンをクリックします。

### 請求メール \{#billing-emails\}

Daytona では、請求書やクレジット残高の消費に関する通知など、重要な請求関連の通知を受け取るための請求メールを設定できます。

1. [Daytona Wallet ↗](https://app.daytona.io/dashboard/billing/wallet) に移動します。
2. 新しいメールアドレスを追加するには、**Add Email** ボタンをクリックします。
3. メールアドレスを入力し、**Add Email** ボタンをクリックします。

確認用メールがそのメールアドレス宛に送信されます。確認が完了すると、そのメールアドレスが請求メールの一覧に追加されます。
