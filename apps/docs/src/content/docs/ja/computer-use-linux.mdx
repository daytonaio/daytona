**コンピューター使用**は、Daytona のサンドボックス内にあるデスクトップ環境をプログラムから制御できる機能です。GUI 操作の自動化やデスクトップアプリのテストに必要な、マウス／キーボード操作、スクリーンショット取得、ディスプレイ制御を提供します。

多くの用途では、このインスタンスタイプの利用を推奨します。

## 一般的なユースケース \{#common-use-cases\}

* **GUIアプリケーションのテスト** - デスクトップアプリの操作を自動化し、ボタンのクリックやフォーム入力、UI動作の検証を行う
* **ビジュアルテストとスクリーンショット** - アプリのスクリーンショットを取得し、UIの状態を比較してビジュアルリグレッションテストを実施する
* **デスクトップ自動化** - 繰り返し作業やGUI経由のファイル管理、複雑なワークフローを自動化する
* **Human-in-the-Loop** - [Daytona Dashboard](https://app.daytona.io/dashboard/sandboxes) からVNCインターフェースに直接アクセスし、自動化スクリプトと併用してデスクトップ環境を手動で操作・対話する

## クイック例 \{#quick-example\}

<Tabs>
  <TabItem label="Python" icon="seti:python">
    ```python
    from daytona import Daytona

    # 環境変数で初期化（推奨）
    daytona = Daytona()
    # または明示的に設定:
    # daytona = Daytona(
    #     api_key=os.getenv('DAYTONA_API_KEY'),
    #     api_url=os.getenv('DAYTONA_API_URL', 'https://app.daytona.io/api')
    # )

    # デフォルトのスナップショットからサンドボックスを作成（デスクトップ環境を含む）
    sandbox = daytona.create()

    # コンピューター使用のプロセスを開始
    result = sandbox.computer_use.start()
    print("コンピューター使用を開始:", result.message)

    # スクリーンショットを取得
    screenshot = sandbox.computer_use.screenshot.take_full_screen()

    # クリックして入力
    sandbox.computer_use.mouse.click(100, 200)
    sandbox.computer_use.keyboard.type('Hello, Linux!')
    sandbox.computer_use.keyboard.hotkey('ctrl+s')
    ```
