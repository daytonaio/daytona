---

**コンピューター使用** は、Daytona のサンドボックス内でデスクトップ環境をプログラムから制御できる機能です。GUI の操作自動化やデスクトップアプリのテストのために、マウス、キーボード、スクリーンショット、ディスプレイ操作を提供します。

macOS 専用アプリケーションとやり取りする必要がある場合は、このインスタンスタイプの利用を推奨します。

:::caution[アルファ版アクセスが必要]
macOS のコンピューター使用は現在 **プライベートアルファ** です。アクセスを希望される方は、[macOS アクセス申請フォーム](https://docs.google.com/forms/d/e/1FAIpQLSc9xlGZ49OjWNkyzDPC9Ip3InMRR0ZXY3tcoD-PFQj3ck6gzQ/viewform?usp=sharing\&ouid=103304973264148733944)にご記入ください。

送信後、当チームが申請内容を審査し、セットアップ手順および提供状況の詳細をご連絡します。
:::

## 一般的なユースケース \{#common-use-cases\}

* **GUIアプリケーションのテスト** - ネイティブなmacOSアプリの操作を自動化し、ボタンのクリック、フォーム入力、UI動作の検証を行う
* **ビジュアルテストとスクリーンショット** - アプリケーションのスクリーンショットを取得し、UIの状態を比較してビジュアル回帰テストを実施する
* **デスクトップ自動化** - 繰り返し発生するmacOSデスクトップ作業やGUIでのファイル管理、複雑なワークフローを自動化する

## はじめに \{#getting-started\}

macOS のコンピューター使用アルファ版へのアクセスが付与されると、Daytona のサンドボックス内で macOS のデスクトップ環境をプログラムから制御できるようになります。この機能には、macOS 環境向けに最適化されたマウス操作、キーボードの自動化、スクリーンショットの取得、ディスプレイ管理が含まれます。
