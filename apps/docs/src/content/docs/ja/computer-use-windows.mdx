**コンピューター使用** は、Daytona のサンドボックス内でデスクトップ環境をプログラムから制御する機能です。GUI 操作の自動化やデスクトップアプリのテストに向けて、マウス、キーボード、スクリーンショット、ディスプレイの操作を提供します。

Windows 専用アプリとの対話が必要な場合は、このインスタンスタイプの使用を推奨します。

:::caution[アルファアクセスが必要]
Windows のコンピューター使用は現在 **プライベートアルファ** です。アクセスをご希望の方は、[アクセス申請フォーム](https://docs.google.com/forms/d/e/1FAIpQLSfoK-77-VpfsMubw8F4f1opCxIL1AyJUgnM0ONYup5hZ0RTvQ/viewform?usp=dialog) にご記入ください。

送信後、当社チームが申請内容を確認し、セットアップ手順および利用可能時期の詳細をご連絡します。
:::

## 一般的なユースケース \{#common-use-cases\}

* **GUIアプリケーションのテスト** - Windowsデスクトップアプリの操作を自動化し、ボタンのクリック、フォーム入力、UIの挙動を検証する
* **ビジュアルテストとスクリーンショット** - アプリのスクリーンショットを取得し、UIの状態を比較してビジュアルリグレッションテストを行う
* **デスクトップ自動化** - Windowsデスクトップでの反復作業やGUI経由のファイル管理、複雑なワークフローを自動化する

## はじめに \{#getting-started\}

Windows のコンピューター使用アルファ版へのアクセスが付与されると、Daytona のサンドボックス内にある Windows デスクトップ環境をプログラムで制御できるようになります。機能には、Windows 環境向けに最適化されたマウス操作、キーボードの自動化、スクリーンショットの取得、ディスプレイ管理が含まれます。
