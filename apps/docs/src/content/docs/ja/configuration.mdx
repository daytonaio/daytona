## コードでの設定 \{#configuration-in-code\}

コード内で環境を設定するには、`DaytonaConfig` クラスを使用します。`DaytonaConfig` クラスは次のパラメータを受け取ります:

* `api_key`: Daytona の [API キー](/docs/api-keys)
* `api_url`: [Daytona API](/docs/ja/tools/api) の URL
* `target`: サンドボックスを作成する対象リージョン（`us` / `eu`）

<Tabs syncKey="language">
  <TabItem label="Python" icon="seti:python">
    ```python
    from daytona import DaytonaConfig

    config = DaytonaConfig(
        api_key="your-api-key",
        api_url="your-api-url",
        target="us"
    )
    ```
  </TabItem>

  <TabItem label="TypeScript" icon="seti:typescript">
    ```typescript
    import { DaytonaConfig } from '@daytonaio/sdk'

    const config: DaytonaConfig = {
      apiKey: 'your-api-key',
      apiUrl: 'your-api-url',
      target: 'us',
    }
    ```
  </TabItem>

  <TabItem label="Ruby" icon="seti:ruby">
    ```ruby
    require 'daytona'

    config = Daytona::Config.new(
      api_key: 'your-api-key',
      api_url: 'your-api-url',
      target: 'us'
    )
    ```
  </TabItem>

  <TabItem label="Go" icon="seti:go">
    ```go
    package main

    import (
        "github.com/daytonaio/daytona/libs/sdk-go/pkg/daytona"
    )

    func main() {
        config := daytona.Config{
            APIKey: "your-api-key",
            APIURL: "your-api-url",
            Target: "us",
        }

        client := daytona.NewClient(&config)

        _ = client
    }
    ```
  </TabItem>

## 環境変数 \{#environment-variables\}

Daytona は設定に環境変数を利用できます。SDK は以下の環境変数を自動的に参照します:

| 変数                   | 説明                                      | 必須     |
| --------------------- | ----------------------------------------- | -------- |
| **`DAYTONA_API_KEY`** | Daytona の API キー。                     | はい     |
| **`DAYTONA_API_URL`** | Daytona API の URL。                      | いいえ   |
| **`DAYTONA_TARGET`**  | サンドボックスを作成する対象の Daytona。 | いいえ   |

### シェル \{#shell\}

シェルで環境変数を設定するには、次のいずれかの方法を使用します:

<Tabs syncKey="shell">
  <TabItem label="Bash/Zsh" icon="seti:shell">
    ```bash
    export DAYTONA_API_KEY=your-api-key
    export DAYTONA_API_URL=https://your-api-url
    export DAYTONA_TARGET=us
    ```
  </TabItem>

  <TabItem label="Windows PowerShell" icon="seti:powershell">
    ```bash
    $env:DAYTONA_API_KEY="your-api-key"
    $env:DAYTONA_API_URL="https://your-api-url"
    $env:DAYTONA_TARGET="us"
    ```
  </TabItem>
</Tabs>

### .env ファイル \{#env-file\}

`.env` ファイルで環境変数を設定するには、次の形式を使用します。

```bash
DAYTONA_API_KEY=YOUR_API_KEY
DAYTONA_API_URL=https://your_api_url
DAYTONA_TARGET=us
```

## デフォルト値 \{#default-values\}

設定がない場合、Daytona は組み込みの既定値を使用します。

| **オプション** | **既定値**                       |
| ---------- | ----------------------------------- |
| API URL    | https://app.daytona.io/api          |
| ターゲット | 組織のデフォルトリージョン         |
