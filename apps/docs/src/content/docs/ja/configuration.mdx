## 設定方法 \{#configuration-methods\}

Daytona SDK では、優先度の高い順に、環境を設定する次の方法をサポートしています:

1. [**コード内での設定**](#configuration-in-code)
2. [**環境変数**](#environment-variables)
3. [**.env ファイル**](#env-file)
4. [**デフォルト値**](#default-values)

## コードでの設定 \{#configuration-in-code\}

Daytona SDK では、Python と TypeScript で `DaytonaConfig` クラスを使って設定を行えます。`DaytonaConfig` クラスは次のパラメータを受け取ります。

* `api_key`: Daytona の [API キー](/docs/api-keys)
* `api_url`: Daytona API の URL
* `target`: サンドボックスを作成する対象の Daytona ターゲット

<Tabs>
  <TabItem label="Python" icon="seti:python">
    ```python
    from daytona import DaytonaConfig

    config = DaytonaConfig(
        api_key="your-api-key",
        api_url="your-api-url",
        target="us"
    )

    ```
  </TabItem>

  <TabItem label="TypeScript" icon="seti:typescript">
    ```typescript
    import { DaytonaConfig } from '@daytonaio/sdk';

    const config: DaytonaConfig = {
        apiKey: "your-api-key",          
        apiUrl: "your-api-url",     
        target: "us"                  
    };
    ```
  </TabItem>
</Tabs>

## 環境変数 \{#environment-variables\}

Daytona SDK は設定に環境変数を利用できます。SDK は自動的に次の環境変数を参照します:

| 変数                  | 説明                                      | 任意     |
| --------------------- | ----------------------------------------- | -------- |
| **`DAYTONA_API_KEY`** | Daytona の API キー                       | いいえ   |
| **`DAYTONA_API_URL`** | Daytona API の URL                        | はい     |
| **`DAYTONA_TARGET`**  | サンドボックスを作成する対象 (Daytona Target) | はい     |

シェルで環境変数を設定します:

<Tabs>
  <TabItem label="Bash/Zsh" icon="seti:shell">
    ```bash
    export DAYTONA_API_KEY=your-api-key
    export DAYTONA_API_URL=https://your-api-url
    export DAYTONA_TARGET=us
    ```
  </TabItem>

  <TabItem label="Windows PowerShell" icon="seti:powershell">
    ```bash
    $env:DAYTONA_API_KEY="your-api-key"
    $env:DAYTONA_API_URL="https://your-api-url"
    $env:DAYTONA_TARGET="us"
    ```
  </TabItem>
</Tabs>

これらの環境変数は `.env` ファイルでも設定できます。

## .env ファイル \{#env-file\}

プロジェクトのルートディレクトリに `.env` ファイルを作成します。

```bash
DAYTONA_API_KEY=your-api-key
DAYTONA_API_URL=https://your-api-url
DAYTONA_TARGET=us
```

SDK は初期化時にこれらの値を自動的に読み込みます。

## 既定値 \{#default-values\}

設定が指定されていない場合、Daytona SDK は組み込みの既定値を使用します：

| オプション | 既定値                         |
|-----------|---------------------------------|
| API URL   | https://app.daytona.io/api      |
| ターゲット | us                              |
