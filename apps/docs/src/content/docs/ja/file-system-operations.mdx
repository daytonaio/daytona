Daytonaは、サンドボックス内の`fs`モジュールを通じて、充実したファイルシステム操作を提供します。

## 基本操作 \{#basic-operations\}

Daytona は、サンドボックス内のファイルシステムを操作するためのメソッドを提供しています。ファイル一覧の取得、ディレクトリの作成、ファイルの読み書きなど、さまざまな操作を実行できます。

ファイル操作は、サンドボックスユーザーのホームディレクトリで作業していることを前提としています（例: `workspace` は `/home/[username]/workspace` を指します）。絶対パスを指定する場合は、先頭に `/` を付けてください。

### ファイルとディレクトリの一覧表示 \{#list-files-and-directories\}

Daytona では、ディレクトリへのパスを指定することで、サンドボックス内のファイルとディレクトリを一覧表示するメソッドを提供します。パスを指定しない場合、メソッドはサンドボックスの作業ディレクトリ内のファイルとディレクトリを一覧表示します。

<Tabs syncKey="language">
  <TabItem label="Python" icon="seti:python">
    ```python
    # List files in a directory
    files = sandbox.fs.list_files("workspace")

    for file in files:
        print(f"Name: {file.name}")
        print(f"Is directory: {file.is_dir}")
        print(f"Size: {file.size}")
        print(f"Modified: {file.mod_time}")
    ```
  </TabItem>

  <TabItem label="TypeScript" icon="seti:typescript">
    ```typescript
    // List files in a directory
    const files = await sandbox.fs.listFiles('workspace')

    files.forEach(file => {
      console.log(`Name: ${file.name}`)
      console.log(`Is directory: ${file.isDir}`)
      console.log(`Size: ${file.size}`)
      console.log(`Modified: ${file.modTime}`)
    })
    ```
  </TabItem>

  <TabItem label="Ruby" icon="seti:ruby">
    ```ruby
    # List directory contents
    files = sandbox.fs.list_files("workspace/data")

    # Print files and their sizes
    files.each do |file|
      puts "#{file.name}: #{file.size} bytes" unless file.is_dir
    end

    # List only directories
    dirs = files.select(&:is_dir)
    puts "Subdirectories: #{dirs.map(&:name).join(', ')}"
    ```
  </TabItem>

  <TabItem label="Go" icon="seti:go">
    ```go
    // List files in a directory
    files, err := sandbox.FileSystem.ListFiles(ctx, "workspace")
    if err != nil {
    	log.Fatal(err)
    }

    for _, file := range files {
    	fmt.Printf("Name: %s\n", file.Name)
    	fmt.Printf("Is directory: %t\n", file.IsDirectory)
    	fmt.Printf("Size: %d\n", file.Size)
    	fmt.Printf("Modified: %s\n", file.ModifiedTime)
    }
    ```
  </TabItem>

  <TabItem label="API" icon="seti:json">
    ```bash
    curl 'https://proxy.app.daytona.io/toolbox/{sandboxId}/files'
    ```
  </TabItem>
</Tabs>

### ディレクトリまたはファイル情報を取得する \{#get-directory-or-file-information\}

Daytona は、ディレクトリまたはファイルへのパスを指定することで、グループ、ディレクトリかどうか、更新日時、モード、名前、所有者、パーミッション、サイズといった情報を取得するためのメソッドを提供します。

<Tabs syncKey="language">

    ```python

    print(f"Modified: {info.mod_time}")
    print(f"Mode: {info.mode}")

    # パスがディレクトリかどうかを確認
    info = sandbox.fs.get_file_info("workspace/data")
    if info.is_dir:
        print("Path is a directory")
    ```
  </TabItem>

  <TabItem label="TypeScript" icon="seti:typescript">

### ファイルのアップロード \{#upload-files\}

Daytona は、サンドボックスに単一ファイルまたは複数ファイルをアップロードするためのメソッドを提供します。

#### 単一ファイルをアップロードする \{#upload-a-single-file\}

Daytona では、アップロードするコンテンツと保存先ファイルのパスを指定して、サンドボックス内に単一ファイルをアップロードするためのメソッドを提供しています。

<Tabs syncKey="language">
  <TabItem label="Python" icon="seti:python">
    ```python
    # 単一ファイルをアップロードする
    with open("local_file.txt", "rb") as f:
        content = f.read()
    sandbox.fs.upload_file(content, "remote_file.txt")
    ```
  </TabItem>

  <TabItem label="TypeScript" icon="seti:typescript">
    ```typescript
    // 単一ファイルをアップロードする
    const fileContent = Buffer.from('Hello, World!')
    await sandbox.fs.uploadFile(fileContent, 'data.txt')
    ```
  </TabItem>

  <TabItem label="Ruby" icon="seti:ruby">
    ```ruby
    # 文字列コンテンツからテキストファイルをアップロードする
    content = "Hello, World!"
    sandbox.fs.upload_file(content, "tmp/hello.txt")

    # ローカルファイルをアップロードする
    sandbox.fs.upload_file("local_file.txt", "tmp/file.txt")

    # バイナリデータをアップロードする
    data = { key: "value" }.to_json
    sandbox.fs.upload_file(data, "tmp/config.json")
    ```
  </TabItem>

  <TabItem label="Go" icon="seti:go">
    ```go

    }

    // またはバイト列データからアップロードする
    content := []byte("Hello, World!")
    err = sandbox.FileSystem.UploadFile(ctx, content, "hello.txt")
    if err != nil {
    	log.Fatal(err)
    }
    ```
  </TabItem>

#### 複数ファイルのダウンロード \{#download-multiple-files\}

Daytona は、ダウンロード対象ファイルのパスを指定することで、サンドボックスから複数のファイルをダウンロードするためのメソッドを提供します。

<Tabs syncKey="language">
  <TabItem label="Python" icon="seti:python">
    ```python
    # 複数ファイルをまとめてダウンロード
    files_to_download = [
        FileDownloadRequest(source="data/file1.txt"), # 保存先なし - メモリにダウンロード
        FileDownloadRequest(source="data/file2.txt", destination="local_file2.txt"), # ローカルファイルにダウンロード
    ]

    results = sandbox.fs.download_files(files_to_download)

    for result in results:
        if result.error:
            print(f"Error downloading {result.source}: {result.error}")
        elif result.result:
            print(f"Downloaded {result.source} to {result.result}")
    ```
  </TabItem>

  <TabItem label="TypeScript" icon="seti:typescript">
    ```typescript
    // 複数ファイルをまとめてダウンロード
    const files = [
      { source: 'data/file1.txt' }, // 保存先なし - メモリにダウンロード
      { source: 'data/file2.txt', destination: 'local_file2.txt' }, // ローカルファイルにダウンロード
    ]

    const results = await sandbox.fs.downloadFiles(files)

    results.forEach(result => {
      if (result.error) {
        console.error(`Error downloading ${result.source}: ${result.error}`)
      } else if (result.result) {
        console.log(`Downloaded ${result.source} to ${result.result}`)
      }
    })
    ```

  <TabItem label="Ruby" icon="seti:ruby">

    ```ruby
    # download_file をそれぞれ呼び出して複数ファイルをダウンロード
    files_to_download = [
      { remote: "data/file1.txt", local: nil },              # メモリにダウンロード
      { remote: "data/file2.txt", local: "local_file2.txt" } # ローカルファイルにダウンロード
    ]

    files_to_download.each do |f|
      if f[:local]
        sandbox.fs.download_file(f[:remote], f[:local])
        puts "Downloaded #{f[:remote]} to #{f[:local]}"
      else
        content = sandbox.fs.download_file(f[:remote])
        puts "Downloaded #{f[:remote]} to memory (#{content.size} bytes)"
      end
    end
    ```

  <TabItem label="Go" icon="seti:go">
    ```go
    // 各ファイルに対して DownloadFile を呼び出して複数ファイルをダウンロード
    filesToDownload := []struct {
    	remotePath string
    	localPath  *string
    }{
    	{"data/file1.txt", nil},                           // メモリにダウンロード
    	{"data/file2.txt", ptrString("local_file2.txt")},  // ローカルファイルにダウンロード
    }

    for _, f := range filesToDownload {
    	content, err := sandbox.FileSystem.DownloadFile(ctx, f.remotePath, f.localPath)
    	if err != nil {
    		fmt.Printf("Error downloading %s: %v\n", f.remotePath, err)
    		continue
    	}
    	if f.localPath == nil {
    		fmt.Printf("Downloaded %s to memory (%d bytes)\n", f.remotePath, len(content))
    	} else {
    		fmt.Printf("Downloaded %s to %s\n", f.remotePath, *f.localPath)
    	}
    }
    ```

### ファイルの検索と置換 \{#find-and-replace-text-in-files\}

Daytona は、検索対象のディレクトリのパスと検索パターンを指定して、ファイル内のテキストの検索および置換を行うメソッドを提供します。

<Tabs syncKey="language">
  <TabItem label="Python" icon="seti:python">
    ```python
    # ファイル内のテキストを検索します。検索対象のディレクトリのパスと検索パターンを指定します
    results = sandbox.fs.find_files(
        path="workspace/src",
        pattern="text-of-interest"
    )
    for match in results:
        print(f"Absolute file path: {match.file}")
        print(f"Line number: {match.line}")
        print(f"Line content: {match.content}")
        print("\n")

    # ファイル内のテキストを置換します
    sandbox.fs.replace_in_files(
        files=["workspace/file1.txt", "workspace/file2.txt"],
        pattern="old_text",
        new_value="new_text"
    )
    ```
  </TabItem>

  <TabItem label="TypeScript" icon="seti:typescript">
    ```typescript

        pattern: "text-of-interest"
    })
    results.forEach(match => {
        console.log('Absolute file path:', match.file)
        console.log('Line number:', match.line)
        console.log('Line content:', match.content)
    })

    // ファイル内のテキストを置換します
    await sandbox.fs.replaceInFiles(
        ["workspace/file1.txt", "workspace/file2.txt"],
        "old_text",
        "new_text"
    )
    ```
  </TabItem>

  <TabItem label="Ruby" icon="seti:ruby">
    ```ruby

      puts "#{match.file}:#{match.line}: #{match.content.strip}"
    end

    # 特定のファイル内で置換を実行します

      files: ["workspace/src/file1.rb", "workspace/src/file2.rb"],
      pattern: "old_function",
      new_value: "new_function"
    )

    # 結果を出力します
    results.each do |result|
      if result.success
        puts "#{result.file}: #{result.success}"
      else
        puts "#{result.file}: #{result.error}"
      end
    end
    ```
  </TabItem>

  <TabItem label="Go" icon="seti:go">
