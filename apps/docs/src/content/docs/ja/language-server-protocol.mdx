Daytona SDKは、Sandboxインスタンスを通じてLanguage Server Protocol（LSP）サポートを提供します。これにより、コード補完、診断などの高度な言語機能が有効になります。

## LSPサーバーの作成

Daytona SDKは、PythonとTypeScriptを使用してLSPサーバーを作成するオプションを提供します。`path_to_project`の開始点は、デフォルトで現在のSandboxユーザーのルートになります - 例：`workspace/project`は`/home/[username]/workspace/project`を意味しますが、`/`で始めることで絶対パスを定義することもできます。

<Tabs>
<TabItem label="Python" icon="seti:python">
```python
from daytona import Daytona, LspLanguageId

# Sandboxを作成

daytona = Daytona()
sandbox = daytona.create()

# Python用のLSPサーバーを作成

lsp_server = sandbox.create_lsp_server(
    language_id=LspLanguageId.PYTHON,
    path_to_project="workspace/project"
)

```
</TabItem>
<TabItem label="TypeScript" icon="seti:typescript">
```typescript
import { Daytona, LspLanguageId } from '@daytonaio/sdk'

// sandboxを作成
const daytona = new Daytona()
const sandbox = await daytona.create({
    language: 'typescript'
})

// TypeScript用のLSPサーバーを作成
const lspServer = await sandbox.createLspServer(
    LspLanguageId.TYPESCRIPT,
    "workspace/project"
)
```

</TabItem>
</Tabs>

## サポートされている言語

Daytona SDKは、PythonとTypeScriptの`LspLanguageId`列挙型を通じて、さまざまな言語のLSPサーバーを作成するオプションを提供します。

<Tabs>
<TabItem label="Python" icon="seti:python">
```python
from daytona import LspLanguageId

# 利用可能な言語ID

LspLanguageId.PYTHON
LspLanguageId.TYPESCRIPT

```
</TabItem>
<TabItem label="TypeScript" icon="seti:typescript">
```typescript
import { LspLanguageId } from '@daytonaio/sdk'

// 利用可能な言語ID
LspLanguageId.PYTHON      
LspLanguageId.TYPESCRIPT    
```

</TabItem>
</Tabs>

- `LspLanguageId.PYTHON`: Python言語サーバー。
- `LspLanguageId.TYPESCRIPT`: TypeScript/JavaScript言語サーバー。

## LSP機能

Daytona SDKは、コード解析と編集のための様々なLSP機能を提供します。

### コード補完

Daytona SDKは、PythonとTypeScriptを使用してファイル内の特定の位置でコード補完を取得するオプションを提供します。

<Tabs>
<TabItem label="Python" icon="seti:python">
```python
completions = lsp_server.completions(
    path="workspace/project/main.py",
    position={"line": 10, "character": 15}
)
print(f"Completions: {completions}")
```
</TabItem>
<TabItem label="TypeScript" icon="seti:typescript">
```typescript
const completions = await lspServer.completions(
    "workspace/project/main.ts",
    { line: 10, character: 15 }
)
console.log('Completions:', completions)
```
</TabItem>
</Tabs>
