---
title: 制限
---

Daytona は、すべての組織における公平な利用と安定性を確保するため、リソース制限を実施しています。組織は次のリソースから成るコンピュートプールにアクセスできます:

- **vCPU** — 利用可能な CPU コア数の合計
- **Memory** — 利用可能な RAM 容量の合計
- **Storage** — 利用可能なディスク容量の合計

これらのリソースはプールされ、実行中のすべてのサンドボックス間で共有されます。\
同時に実行できるサンドボックスの数は、各サンドボックスが使用する CPU、RAM、ストレージの量に応じて決まります。リソース使用量の設定と見積もり方法は、[サンドボックス管理のドキュメント](https://www.daytona.io/docs/sandbox-management/)を参照してください。

## ティアと上限引き上げ

組織は認証状況に基づいて自動的にティアに割り当てられます。\
以下のステップを完了すると、より高い上限を利用できるようになります:

| ティア | リソース（vCPU / RAM / Storage） | アクセス要件                                                                                                             |
| ------ | -------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| ティア1 | 10 / 10GiB / 30GiB               | メール認証済み                                                                                                           |
| ティア2 | 100 / 200GiB / 300GiB            | クレジットカード登録、$25のチャージ、[GitHub 連携](https://www.daytona.io/docs/linked-accounts#how-to-link-an-account)。 |
| ティア3 | 250 / 500GiB / 2000GiB           | 企業メール認証済み、電話認証済み、$500のチャージ。                                                                        |
| ティア4 | 500 / 1000GiB / 5000GiB          | 30日ごとに$2000のチャージ。                                                                                              |
| カスタム | カスタム上限                     | [support@daytona.io](mailto:support@daytona.io) までご連絡ください                                                       |

より高いティアの条件を満たしたら、[Dashboard](https://app.daytona.io/dashboard/limits) の「Upgrade」ボタンをクリックしてアップグレードしてください。

:::tip

### 利用状況を動的に管理

サンドボックスの状態による利用可能リソースへの影響は次のとおりです:

- **Running:** Compute、メモリ、ストレージを使用します。
- **Stopped:** Compute とメモリは解放されますが、ストレージは使用します。
- **Archived:** データをコールドストレージに移し、すべてのリソースを解放します。
- **Deleted:** 完全に削除され、すべてのリソースが解放されます。

:::

現在の利用状況と上限は [Dashboard](https://app.daytona.io/dashboard/limits) で確認できます。

## さらに必要ですか？

より高い、または特別な上限が必要な場合は、[support@daytona.io](mailto:support@daytona.io) までお問い合わせください。
