---
title: 連携アカウント
---

Daytona は、さまざまなアイデンティティプロバイダーのユーザーアカウント連携をサポートしています。現時点で対応しているプロバイダーは次のとおりです:

* Google
* GitHub

連携済みのいずれかのプロバイダーを使用して、Daytona アカウントにログインできます。

:::tip

#### さらに高い利用上限を開放 \{#unlock-higher-usage-limits\}

GitHub アカウントの連携は、Tier 2 への自動アップグレード要件の一つです。

:::

## アカウントのリンク／解除 \{#linkingunlinking-accounts\}

アカウントをリンクまたは解除するには:

1. Daytona ダッシュボードの [Linked Accounts](https://app.daytona.io/dashboard/user/linked-accounts) ページにアクセスします。
2. 管理したいプロバイダーの横にある「Link Account」または「Unlink」ボタンをクリックします。

## ほかに必要なものはありますか？ \{#need-more\}

他のアイデンティティプロバイダーへの対応が必要な場合は、[support@daytona.io](mailto:support@daytona.io) までご連絡ください。
