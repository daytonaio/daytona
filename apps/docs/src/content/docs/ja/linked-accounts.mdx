---
title: 連携アカウント
---

Daytona は、さまざまなアイデンティティプロバイダーのユーザーアカウント連携をサポートしています。現時点で対応しているプロバイダーは次のとおりです:

* Google
* GitHub

:::tip
GitHub アカウントは、[組織を自動的に **Tier 2** にアップグレードする](/docs/ja/limits#tiers--rate-limit-increases)ための要件の一つです。
:::

## アカウントを連携する \{#link-account\}

1. [Daytona Dashboard ↗](https://app.daytona.io/dashboard/user/account-settings) のアカウント設定にアクセスします。
2. 連携したいプロバイダーの横にある **Link Account** ボタンをクリックします。
3. 画面の指示に従ってアカウントを連携します。

## アカウントの連携解除 \{#unlink-account\}

1. [Daytona Dashboard ↗](https://app.daytona.io/dashboard/user/account-settings) のアカウント設定ページにアクセスします。
2. 連携を解除したいプロバイダーの横にある **Unlink** ボタンをクリックします。
3. 画面の指示に従ってアカウントの連携を解除します。
