ポート範囲 `3000-9999` で HTTP トラフィックを待ち受けるプロセスは、プレビューリンクで確認できます。

プレビューリンクの形式は「ポート + サンドボックスID」の組み合わせです。例:
`https://3000-sandbox-123456.proxy.daytona.work`

サンドボックスの `public` プロパティが `true` の場合、これらのリンクは公開アクセス可能です。そうでない場合、プレビューリンクはそのサンドボックスの組織のユーザーのみが利用できます。

プログラムからアクセスする場合（例: curl）は、プレビューURLにアクセスするときに認可トークンを渡すため、X-Daytona-Preview-Token ヘッダーを使用してください:

```bash
curl -H "x-daytona-preview-token: vg5c0ylmcimr8b_v1ne0u6mdnvit6gc0" \
https://3000-sandbox-123456.proxy.daytona.work
```

または、追加ヘッダーをサポートしないランタイム（例: ブラウザー環境）では、DAYTONA\_SANDBOX\_AUTH\_KEY クエリパラメータに認可トークンを指定してください:

```bash
curl "https://3000-sandbox-123456.proxy.daytona.work?DAYTONA_SANDBOX_AUTH_KEY=vg5c0ylmcimr8b_v1ne0u6mdnvit6gc0"
```

特定のポートに対するプレビューリンクと認可トークンは、SDK メソッドで取得できます:

<Tabs>
  <TabItem label="Python" icon="seti:python">
    ```python

    preview_info = sandbox.get_preview_link(3000)

    print(f"Preview link url: {preview_info.url}")
    print(f"Preview link token: {preview_info.token}")

    ```
  </TabItem>

  <TabItem label="TypeScript" icon="seti:typescript">
    ```typescript

    const previewInfo = await sandbox.getPreviewUrl(3000);

    console.log(`Preview link url: ${previewInfo.url}`);
    console.log(`Preview link token: ${previewInfo.token}`);

    ```
  </TabItem>
</Tabs>

:::tip
Daytona の URL の代わりに自分のドメイン配下でプレビューを提供したい場合は、[カスタムドメイン/認証](/docs/ja/custom-domain-authentication)を参照してください。
:::
