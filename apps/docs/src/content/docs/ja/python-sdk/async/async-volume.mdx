## ボリューム \{#volume\}

```python
class Volume(VolumeDto)
```

サンドボックス向けの共有ストレージである Daytona のボリュームを表します。

**属性**:

* `id` *StrictStr* - ボリュームの一意の識別子。
* `name` *StrictStr* - ボリューム名。
* `organization_id` *StrictStr* - ボリュームの属する組織 ID。
* `state` *StrictStr* - ボリュームの状態。
* `created_at` *StrictStr* - ボリュームの作成日時。
* `updated_at` *StrictStr* - ボリュームの最終更新日時。
* `last_used_at` *StrictStr* - ボリュームの最終使用日時。

## AsyncVolumeService（非同期ボリュームサービス） \{#asyncvolumeservice\}

```python
class AsyncVolumeService()
```

Daytona のボリュームを管理するサービス。ボリュームの一覧表示、取得、作成、削除に使用できます。

#### AsyncVolumeService.list \{#asyncvolumeservicelist\}

```python
async def list() -> List[Volume]
```

すべてのボリュームを一覧表示します。

**戻り値**:

* `List[Volume]` - すべてのボリュームのリスト。

**例**:

```python
async with AsyncDaytona() as daytona:
    volumes = await daytona.volume.list()
    for volume in volumes:
        print(f"{volume.name} ({volume.id})")
```

#### AsyncVolumeService.get \{#asyncvolumeserviceget\}

```python
async def get(name: str, create: bool = False) -> Volume
```

名前でボリュームを取得します。

**引数**:

* `name` *str* - 取得するボリュームの名前。
* `create` *bool* - True の場合、存在しない場合は新しいボリュームを作成します。

**戻り値**:

* `Volume` - ボリュームのオブジェクト。

**例**:

```python
async with AsyncDaytona() as daytona:
    volume = await daytona.volume.get("test-volume-name", create=True)
    print(f"{volume.name} ({volume.id})")
```

#### AsyncVolumeService.create \{#asyncvolumeservicecreate\}

```python
async def create(name: str) -> Volume
```

新しいボリュームを作成します。

**引数**:

* `name` *str* - 作成するボリューム名。

**返り値**:

* `Volume` - Volume オブジェクト。

**例**:

```python
async with AsyncDaytona() as daytona:
    volume = await daytona.volume.create("test-volume")
    print(f"{volume.name} ({volume.id}); 状態: {volume.state}")
```

#### AsyncVolumeService.delete \{#asyncvolumeservicedelete\}

```python
async def delete(volume: Volume) -> None
```

ボリュームを削除します。

**引数**:

* `volume` *ボリューム* - 削除するボリューム。

**例**:

```python
async with AsyncDaytona() as daytona:
    volume = await daytona.volume.get("test-volume")
    await daytona.volume.delete(volume)
    print("ボリュームが削除されました")
```
