## Chart

```python
class Chart()
```

matplotlibからのメタデータを持つチャートを表します。

**属性**:

- `type` _ChartType_ - チャートのタイプ
- `title` _str_ - チャートのタイトル
- `elements` _List[Any]_ - チャートの要素
- `png` _Optional[str]_ - base64でエンコードされたチャートのPNG表現


## ChartType

```python
class ChartType(str, Enum)
```

チャートタイプ

**Enum Members**:
    - `LINE` ("line")
    - `SCATTER` ("scatter")
    - `BAR` ("bar")
    - `PIE` ("pie")
    - `BOX_AND_WHISKER` ("box_and_whisker")
    - `COMPOSITE_CHART` ("composite_chart")
    - `UNKNOWN` ("unknown")

## Chart2D

```python
class Chart2D(Chart)
```

メタデータを持つ2Dチャートを表します。

**属性**:

- `x_label` _Optional[str]_ - x軸のラベル
- `y_label` _Optional[str]_ - y軸のラベル

## PointData

```python
class PointData()
```

2Dチャートの点を表します。

**属性**:

- `label` _str_ - 点のラベル
- `points` _List[Tuple[Union[str, float], Union[str, float]]]_ - チャートの点

## PointChart

```python
class PointChart(Chart2D)
```

メタデータを持つポイントチャートを表します。

**属性**:

- `x_ticks` _List[Union[str, float]]_ - x軸の目盛り
- `x_tick_labels` _List[str]_ - x軸のラベル
- `x_scale` _str_ - x軸のスケール
- `y_ticks` _List[Union[str, float]]_ - y軸の目盛り
- `y_tick_labels` _List[str]_ - y軸のラベル
- `y_scale` _str_ - y軸のスケール
- `elements` _List[PointData]_ - チャートのポイント

## BoxAndWhiskerData

```python
class BoxAndWhiskerData()
```

ボックスプロット（箱ひげ図）チャートにおけるボックスとひげを表します。

**属性**:

- `label` _str_ - ボックスとひげのラベル
- `min` _float_ - ボックスとひげの最小値
- `first_quartile` _float_ - ボックスとひげの第1四分位数
- `median` _float_ - ボックスとひげの中央値
- `third_quartile` _float_ - ボックスとひげの第3四分位数
- `max` _float_ - ボックスとひげの最大値
- `outliers` _List[float]_ - ボックスとひげの外れ値

## CompositeChart

```python
class CompositeChart(Chart)
```

メタデータを持つ複合チャートを表します。複合チャートは複数のチャート（サブプロット）を含むチャートです。

**属性**:

- `type` _ChartType_ - チャートのタイプ
- `elements` _List[Chart]_ - 複合チャートのチャート（サブプロット）
