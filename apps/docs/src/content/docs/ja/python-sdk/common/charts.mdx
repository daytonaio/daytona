## グラフ \{#chart\}

```python
class Chart()
```

matplotlib のメタデータを含むチャートを表します。

**属性**:

* `type` *ChartType* - チャートのタイプ
* `title` *str* - チャートのタイトル
* `elements` *List[Any]* - チャートの要素
* `png` *Optional[str]* - base64 エンコードされたチャートの PNG 表現

## ChartType \{#charttype\}

```python
class ChartType(str, Enum)
```

チャートの種類

**列挙メンバー**:

* `LINE` (&quot;line&quot;)
  * `SCATTER` (&quot;scatter&quot;)
  * `BAR` (&quot;bar&quot;)
  * `PIE` (&quot;pie&quot;)
  * `BOX_AND_WHISKER` (&quot;box&#95;and&#95;whisker&quot;)
  * `COMPOSITE_CHART` (&quot;composite&#95;chart&quot;)
  * `UNKNOWN` (&quot;unknown&quot;)

## Chart2D \{#chart2d\}

```python
class Chart2D(Chart)
```

メタデータを含む2Dチャートを表します。

**属性**:

* `x_label` *Optional[str]* - x軸のラベル
* `y_label` *Optional[str]* - y軸のラベル

## PointData \{#pointdata\}

```python
class PointData()
```

2D チャート上の点を表します。

**属性**:

* `label` *str* - 点のラベル
* `points` *List[Tuple[Union[str, float], Union[str, float]]]* - チャート上の座標点のリスト

## PointChart \{#pointchart\}

```python
class PointChart(Chart2D)
```

メタデータを含む散布図を表します。

**属性**:

* `x_ticks` *List[Union[str, float]]* - x軸の目盛
* `x_tick_labels` *List[str]* - x軸の目盛ラベル
* `x_scale` *str* - x軸のスケール
* `y_ticks` *List[Union[str, float]]* - y軸の目盛
* `y_tick_labels` *List[str]* - y軸の目盛ラベル
* `y_scale` *str* - y軸のスケール
* `elements` *List[PointData]* - 図のプロット点

## CompositeChart \{#compositechart\}

```python
class CompositeChart(Chart)
```

メタデータを伴う複合チャートを表します。複合チャートは、複数のチャート（サブプロット）から構成されるチャートです。

**属性**:

* `type` *ChartType* - チャートのタイプ
* `elements` *List[Chart]* - 複合チャートを構成するチャート（サブプロット）
