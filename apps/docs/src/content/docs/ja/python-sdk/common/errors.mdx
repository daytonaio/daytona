## DaytonaError \{#daytonaerror\}

```python
class DaytonaError(Exception)
```

Daytona SDK の基底エラー。

#### DaytonaError.&#95;&#95;init&#95;&#95; \{#daytonaerror&#95;&#95;init&#95;&#95;\}

```python
def __init__(message: str,
             status_code: Optional[int] = None,
             headers: Optional[dict] = None)
```

Daytona の初期化エラー。

**引数**:

* `message` - エラーメッセージ
* `status_code` - 利用可能な場合の HTTP ステータスコード
* `headers` - 利用可能な場合のレスポンスヘッダー
