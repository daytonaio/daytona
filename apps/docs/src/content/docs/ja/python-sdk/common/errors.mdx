## DaytonaError \{#daytonaerror\}

```python
class DaytonaError(Exception)
```

Daytona SDK の基底エラー。

**属性**:

* `message` *str* - エラーメッセージ
* `status_code` *int | None* - 利用可能な場合の HTTP ステータスコード
* `headers` *dict[str, Any]* - レスポンスヘッダー

#### DaytonaError.**init** \{#daytonaerror__init__\}

```python
def __init__(message: str,
             status_code: int | None = None,
             headers: Mapping[str, Any] | None = None)
```

Daytona の初期化エラー。

**引数**:

* `message` *str* - エラーメッセージ
* `status_code` *int | None* - 利用可能な場合の HTTP ステータスコード
* `headers` *Mapping[str, Any] | None* - 利用可能な場合のレスポンスヘッダー
