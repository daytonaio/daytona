## ボリューム \{#volume\}

```python
class Volume(VolumeDto)
```

サンドボックス向けの共有ストレージである Daytona のボリュームを表します。

**属性**:

* `id` *StrictStr* - ボリュームの一意の識別子。
* `name` *StrictStr* - ボリューム名。
* `organization_id` *StrictStr* - ボリュームの組織 ID。
* `state` *StrictStr* - ボリュームの状態。
* `created_at` *StrictStr* - ボリュームの作成日時。
* `updated_at` *StrictStr* - ボリュームの最終更新日時。
* `last_used_at` *StrictStr* - ボリュームの最終使用日時。

## VolumeService \{#volumeservice\}

```python
class VolumeService()
```

Daytona のボリュームを管理するサービス。ボリュームの一覧表示、取得、作成、削除に使用できます。

#### VolumeService.list（ボリュームの一覧） \{#volumeservicelist\}

```python
def list() -> List[Volume]
```

すべてのボリュームを一覧表示します。

**戻り値**:

* `List[Volume]` - すべてのボリュームのリスト。

**例**:

```python
daytona = Daytona()
volumes = daytona.volume.list()
for volume in volumes:
    print(f"{volume.name} ({volume.id})")
```

#### VolumeService.get \{#volumeserviceget\}

```python
def get(name: str, create: bool = False) -> Volume
```

名前を指定してボリュームを取得します。

**引数**:

* `name` *str* - 取得するボリュームの名前。
* `create` *bool* - True の場合、存在しない場合は新しいボリュームを作成します。

**戻り値**:

* `Volume` - Volume オブジェクト。

**例**:

```python
daytona = Daytona()
volume = daytona.volume.get("test-volume-name", create=True)
print(f"{volume.name} ({volume.id})")
```

#### VolumeService.create \{#volumeservicecreate\}

```python
def create(name: str) -> Volume
```

新しいボリュームを作成します。

**引数**:

* `name` *str* - 作成するボリュームの名前。

**戻り値**:

* `Volume` - Volume オブジェクト。

**例**:

```python
daytona = Daytona()
volume = daytona.volume.create("test-volume")
print(f"{volume.name} ({volume.id}); 状態: {volume.state}")
```

#### VolumeService.delete（ボリュームを削除） \{#volumeservicedelete\}

```python
def delete(volume: Volume) -> None
```

ボリュームを削除します。

**引数**:

* `volume` *ボリューム* - 削除するボリューム。

**例**:

```python
daytona = Daytona()
volume = daytona.volume.get("test-volume")
daytona.volume.delete(volume)
print("ボリュームが削除されました")
```

## VolumeMount \{#volumemount\}

```python
class VolumeMount(ApiVolumeMount, AsyncApiVolumeMount)
```

サンドボックス用のボリュームマウント設定を表します。

**属性**:

* `volume_id` *str* - マウントするボリュームのID。
* `mount_path` *str* - サンドボックス内でボリュームがマウントされるパス。
* `subpath` *str, optional* - ボリューム内でマウントするS3のサブパス/プレフィックス（任意）。
  指定した場合、このプレフィックスのみがアクセス可能になります。省略した場合は、
  ボリューム全体がマウントされます。
