## ComputerUse \{#computeruse\}

新しい ComputerUse インスタンスを初期化します。

### コンストラクタ \{#constructors\}

#### new ComputerUse() \{#new-computeruse\}

```ruby
def initialize(sandbox_id:, toolbox_api:, otel_state:)

```

新しい ComputerUse インスタンスを初期化します。

**パラメーター**:

* `sandbox_id` *String* - サンドボックスの ID
* `toolbox_api` *DaytonaApiClient:ToolboxApi* - サンドボックス操作用の API クライアント
* `otel_state` *Daytona:OtelState, nil* -

**戻り値**:

* `ComputerUse` - 新しい ComputerUse インスタンスを返します

#### start() \{#start\}

```ruby
def start()

```

すべてのコンピューター使用関連プロセス（Xvfb、xfce4、x11vnc、novnc）を開始します。

**戻り値**:

* `DaytonaApiClient:ComputerUseStartResponse` - コンピューター使用の開始レスポンス

**スローされる例外**:

* `Daytona:Sdk:Error` - 操作が失敗した場合

**使用例:**

```ruby
result = sandbox.computer_use.start
puts "コンピューター使用プロセスが開始されました: #{result.message}"

```

#### stop() \{#stop\}

```ruby
def stop()

```

すべてのコンピューター使用プロセスを停止します。

**戻り値**:

* `DaytonaApiClient:ComputerUseStopResponse` - コンピューター使用停止レスポンス

**例外**:

* `Daytona:Sdk:Error` - 操作が失敗した場合に送出されます

**使用例:**

```ruby
result = sandbox.computer_use.stop
puts "コンピューター使用プロセスが停止しました: #{result.message}"

```

#### status() \{#status\}

```ruby
def status()

```

すべてのコンピューター使用プロセスのステータスを取得します。

**戻り値**:

* `DaytonaApiClient:ComputerUseStatusResponse` - すべての VNC デスクトップ プロセスのステータス情報

**例外**:

* `Daytona:Sdk:Error` - 操作が失敗した場合に発生します

**使用例:**

```ruby
response = sandbox.computer_use.get_status
puts "コンピューター使用ステータス: #{response.status}"

```

#### get_process_status() \{#get_process_status\}

```ruby
def get_process_status(process_name:)

```

特定の VNC プロセスのステータスを取得します。

**パラメータ**:

* `process_name` *String* - 確認するプロセスの名前

**戻り値**:

* `DaytonaApiClient:ProcessStatusResponse` - 特定のプロセスに関するステータス情報

**例外**:

* `Daytona:Sdk:Error` - 操作が失敗した場合に発生します

**使用例:**

```ruby
xvfb_status = sandbox.computer_use.get_process_status("xvfb")
no_vnc_status = sandbox.computer_use.get_process_status("novnc")

```
