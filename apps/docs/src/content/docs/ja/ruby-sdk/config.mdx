## Config \{#config\}

新しい Daytona::Config オブジェクトを表すメインクラスです。

### コンストラクター \{#constructors\}

#### new Config() \{#new-config\}

```ruby
def initialize(api_key:, jwt_token:, api_url:, organization_id:, target:, _experimental:)

```

新しい Daytona::Config オブジェクトを初期化します。

**パラメーター**:

* `api_key` *String, nil* - Daytona の API キー。デフォルトは ENV[&#39;DAYTONA&#95;API&#95;KEY&#39;] です。
* `jwt_token` *String, nil* - Daytona の JWT トークン。デフォルトは ENV[&#39;DAYTONA&#95;JWT&#95;TOKEN&#39;] です。
* `api_url` *String, nil* - Daytona の API URL。デフォルトは ENV[&#39;DAYTONA&#95;API&#95;URL&#39;] または Daytona::Config::API&#95;URL です。
* `organization_id` *String, nil* - Daytona の組織 ID。デフォルトは ENV[&#39;DAYTONA&#95;ORGANIZATION&#95;ID&#39;] です。
* `target` *String, nil* - Daytona のターゲット。デフォルトは ENV[&#39;DAYTONA&#95;TARGET&#39;] です。
* `_experimental` *Hash, nil* - 実験的な設定オプション。

**戻り値**:

* `Config` - 新しい Config インスタンス
