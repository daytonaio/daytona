## Daytona \{#daytona\}

Daytona SDK の Daytona クラスです。

### コンストラクター \{#constructors\}

#### new Daytona() \{#new-daytona\}

```ruby
def initialize(config)

```

**パラメーター**:

* `config` *Daytona:Config* - 設定オプション。デフォルト値は Daytona::Config.new

**戻り値**:

* `Daytona` - 新しい Daytona のインスタンス

#### close() \{#close\}

```ruby
def close()

```

OTel プロバイダーをシャットダウンし、保留中のテレメトリデータをすべてフラッシュします。

**戻り値**:

* `void`

#### create() \{#create\}

```ruby
def create(params, on_snapshot_create_logs:)

```

指定されたパラメータでサンドボックスを作成します

**パラメーター**:

* `params` *Daytona:CreateSandboxFromSnapshotParams, Daytona:CreateSandboxFromImageParams, Nil* - サンドボックス作成用のパラメータ

**戻り値**:

* `Daytona:Sandbox` - 作成されたサンドボックス

**例外**:

* `Daytona:Sdk:Error` - auto&#95;stop&#95;interval または auto&#95;archive&#95;interval が負の値の場合

#### start() \{#start\}

```ruby
def start(sandbox, timeout)

```

サンドボックスを起動し、準備が完了するまで待機します。

**パラメータ**:

* `sandbox` *Daytona:Sandbox* -
* `timeout` *Numeric* - 最大待機時間（秒）。指定しない場合は 60 秒です。

**戻り値**:

* `void`

#### stop() \{#stop\}

```ruby
def stop(sandbox, timeout)

```

サンドボックスを停止し、停止されるまで待機します。

**パラメーター**:

* `sandbox` *Daytona:Sandbox* -
* `timeout` *Numeric* - 最大待機時間（秒）。省略時は 60 秒です。

**戻り値**:

* `void`
