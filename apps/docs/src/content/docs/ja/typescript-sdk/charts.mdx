## ChartType

チャートタイプ

**列挙型メンバー**:

- `BAR` ("bar")
- `LINE` ("line")
- `PIE` ("pie")
- `SCATTER` ("scatter")
- `UNKNOWN` ("unknown")

## parseChart()

```ts
function parseChart(data: any): Chart
```

**パラメータ**:

- `data` _any_


**戻り値**:

- `Chart`

***


## BarChart

```ts
type BarChart = Chart2D & {
  elements: BarData[];
  type: BAR;
};
```

メタデータを持つ棒グラフを表します。

**型宣言**:

- `elements` _BarData\[\]_ - チャートの棒
- `type` _BAR_ - チャートの種類
    


## BarData

```ts
type BarData = {

  value: string;
};
```

棒グラフの棒を表します。

**型宣言**:

- `group` _string_ - 棒のグループ
- `label` _string_ - 棒のラベル
- `value` _string_ - 棒の値
    


## BoxAndWhiskerChart

```ts
type BoxAndWhiskerChart = Chart2D & {

```

**型宣言**:

- `elements` _BoxAndWhiskerData\[\]_ - チャートのボックスとウィスカー
- `type` _BOX\_AND\_WHISKER_ - チャートの種類
    


## BoxAndWhiskerData

```ts
type BoxAndWhiskerData = {
  first_quartile: number;
  label: string;
  max: number;
  median: number;
  min: number;
  outliers: number[];
};
```

ボックスアンドウィスカーチャートにおけるボックスアンドウィスカーを表します。

**型宣言**:

- `first\_quartile` _number_ - ボックスアンドウィスカーの第1四分位数
- `label` _string_ - ボックスアンドウィスカーのラベル
- `max` _number_ - ボックスアンドウィスカーの第3四分位数
- `median` _number_ - ボックスアンドウィスカーの中央値
- `min` _number_ - ボックスアンドウィスカーの最小値
- `outliers` _number\[\]_


## Chart

```ts
type Chart = {
  elements: any[];
  png: string;
  title: string;
  type: ChartType;
};
```

matplotlibからのメタデータを持つチャートを表します。

**型宣言**:

- `elements` _any\[\]_ - チャートの要素
- `png?` _string_ - base64でエンコードされたチャートのPNG表現
- `title` _string_ - チャートのタイトル
- `type` _ChartType_ - チャートの種類

## PointChart

```ts
type PointChart = Chart2D & {
  elements: PointData[];
  x_scale: string;
  x_tick_labels: string[];
  x_ticks: (number | string)[];
  y_scale: string;
  y_tick_labels: string[];
  y_ticks: (number | string)[];
};
```

メタデータを持つポイントチャートを表します。

**型宣言**:

- `elements` _PointData\[\]_ - チャートのポイント
- `x\_scale` _string_ - x軸のスケール
- `x\_tick\_labels` _string\[\]_ - x軸のラベル
- `x\_ticks` _\(number \| string\)\[\]_ - x軸の目盛り
- `y\_scale` _string_ - y軸のスケール
- `y\_tick\_labels` _string\[\]_ - y軸のラベル
- `y\_ticks` _\(number \| string\)\[\]_ - y軸の目盛り
