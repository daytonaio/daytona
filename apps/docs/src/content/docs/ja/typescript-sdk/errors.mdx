## DaytonaError \{#daytonaerror\}

Daytona SDK の基底エラークラス。

**プロパティ**:

* `headers?` *AxiosHeaders* - 存在する場合のレスポンスヘッダー
* `statusCode?` *number* - 存在する場合の HTTP ステータスコード

**継承:**

* `Error`

### 次のクラスにより拡張されます \{#extended-by\}

* `DaytonaNotFoundError`
* `DaytonaRateLimitError`
* `DaytonaTimeoutError`

### コンストラクタ \{#constructors\}

#### new DaytonaError() \{#new-daytonaerror\}

```ts
new DaytonaError(
   message: string, 
   statusCode?: number, 
   headers?: AxiosHeaders): DaytonaError
```

**パラメータ**:

* `message` *string*
* `statusCode?` *number*
* `headers?` *AxiosHeaders*

**戻り値**:

* `DaytonaError`

##### オーバーライド \{#overrides\}

```ts
Error.constructor
```
