## ExecuteResponse \{#executeresponse\}

コマンド実行のレスポンス。

**プロパティ**:

* `artifacts?` *ExecutionArtifacts* - コマンド実行で生成された実行アーティファクト
* `exitCode` *number* - コマンド実行の終了コード
* `result` *string* - コマンド実行の出力

**継承:**

* `ExecuteResponse`

## ExecutionArtifacts \{#executionartifacts\}

コマンド実行で生成されたアーティファクト。

**プロパティ**:

* `charts?` *Chart[]* - matplotlib 由来のチャートメタデータのリスト
* `stdout` *string* - コマンドの標準出力。`ExecuteResponse` の `result` と同一
