## VolumeService \{#volumeservice\}

Daytona のボリュームを管理するためのサービス。

このサービスは、ボリュームの一覧表示、取得、作成、および削除を行うメソッドを提供します。

ボリュームは、オプションの subpath パラメーターを指定してサンドボックスにマウントでき、
ボリューム内の特定の S3 プレフィックスのみをマウントすることができます。subpath が指定されていない場合は、
ボリューム全体がマウントされます。

### コンストラクタ \{#constructors\}

#### new VolumeService() \{#new-volumeservice\}

```ts
new VolumeService(volumesApi: VolumesApi): VolumeService
```

**パラメータ**:

* `volumesApi` *VolumesApi*

**戻り値**:

* `VolumeService`

### メソッド \{#methods\}

#### create() \{#create\}

```ts
create(name: string): Promise<Volume>
```

指定された名前の新しい Volume を作成します。

**Parameters**:

* `name` *string* - 新しい Volume の名前

**Returns**:

* `Promise<Volume>` - 作成された新しい Volume

**Throws**:

Volume を作成できなかった場合にスローされます

**Example:**

```ts
const daytona = new Daytona();
const volume = await daytona.volume.create("my-data-volume");
console.log(`ボリューム ${volume.name} を作成しました (ID: ${volume.id})`);
```

***

#### delete() \{#delete\}

```ts
delete(volume: Volume): Promise<void>
```

ボリュームを削除します。

**パラメーター**:

* `volume` *Volume* - 削除するボリューム

**戻り値**:

* `Promise<void>`

**スローされる例外**:

ボリュームが存在しない、または削除できない場合にスローされます

**例:**

```ts
const daytona = new Daytona();
const volume = await daytona.volume.get("volume-name");
await daytona.volume.delete(volume);
console.log("ボリュームが正常に削除されました");
```

***

#### get() \{#get\}

```ts
get(name: string, create: boolean): Promise<Volume>
```

名前を指定してボリュームを取得します。

**パラメータ**:

* `name` *string* - 取得するボリュームの名前
* `create` *boolean = false* - ボリュームが存在しない場合に作成するかどうか

**返り値**:

* `Promise<Volume>` - 指定されたボリューム

**スロー**:

ボリュームが存在しない、またはアクセスできない場合にスローされます

**例:**

```ts
const daytona = new Daytona();
const volume = await daytona.volume.get("volume-name", true);
console.log(`Volume ${volume.name} is in state ${volume.state}`);
```

***

#### list() \{#list\}

```ts
list(): Promise<Volume[]>
```

利用可能なすべてのボリュームを一覧表示します。

**戻り値**:

* `Promise<Volume[]>` - ユーザーがアクセスできるすべてのボリュームの一覧

**例:**

```ts
const daytona = new Daytona();
const volumes = await daytona.volume.list();
console.log(`Found ${volumes.length} volumes`);
volumes.forEach(vol => console.log(`${vol.name} (${vol.id})`));
```

## ボリューム \{#volume\}

```ts
type Volume = VolumeDto & {
  __brand: "Volume";
};
```

Daytona のボリュームを表します。サンドボックス間で共有されるストレージボリュームです。

**型定義**:

* `\_\_brand` *&quot;Volume&quot;*
