## Chart

```ts

  elements: any[]
  png: string

  type: ChartType
}
```

Represents a chart with metadata from matplotlib.

**Type declaration**:

- `elements` _any\[\]_ - The elements of the chart
- `png?` _string_ - The PNG representation of the chart encoded in base64

- `type` _ChartType_ - The type of chart
