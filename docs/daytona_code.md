## daytona code

Open a workspace in your preferred IDE

```
daytona code [WORKSPACE] [PROJECT] [flags]
```

### Options

```
  -i, --ide string   Specify the IDE (vscode, browser, cursor, ssh, clion, goland, intellij, phpstorm, pycharm, rider, rubymine, webstorm)
```
