## daytona logs

View logs for a workspace or project

```
daytona logs [WORKSPACE] [PROJECT] [flags]
```

### Options

```
  -a, --all         Show logs for workspace and project together
  -f, --follow      Follow logs
  -w, --workspace   Show logs for workspace only
```
